# 一键启动脚本说明

## 主要启动脚本

### 1. `start.sh` - 推荐使用 ⭐
最简单的启动方式，直接运行节点：
```bash
cd /home/pc/realsense
./start.sh
```

**特点：**
- 自动检查并启动roscore
- 自动加载ROS环境
- 前台运行，方便查看日志
- 按 Ctrl+C 停止

### 2. `start_with_viewer.sh`
启动节点并自动打开图像查看器：
```bash
./start_with_viewer.sh
```

**特点：**
- 自动启动roscore
- 启动RealSense节点
- 自动打开人脸检测图像查看器
- 后台运行多个进程

### 3. `start_face_detection.sh`
完整版启动脚本（使用gnome-terminal，需要GUI环境）：
```bash
./start_face_detection.sh
```

**特点：**
- 自动打开多个终端窗口
- 分别显示roscore、节点、图像查看器
- 适合桌面环境使用

### 4. `start_simple.sh`
简化版（适合SSH或非GUI环境）：
```bash
./start_simple.sh
```

**特点：**
- 前台运行
- 不依赖GUI
- 需要手动在另一个终端启动roscore

## 查看结果

启动节点后，在另一个终端运行：

```bash
cd /home/pc/realsense
source devel/setup.bash

# 查看人脸检测结果
rosrun image_view image_view image:=/camera/face_detection/image_raw

# 或查看原始图像
rosrun image_view image_view image:=/camera/color/image_raw
```

## 快速测试

```bash
# 1. 启动节点
./start.sh

# 2. 在另一个终端查看话题
source devel/setup.bash
rostopic list

# 3. 查看人脸检测图像
rosrun image_view image_view image:=/camera/face_detection/image_raw
```

## 停止节点

- 如果使用 `start.sh` 或 `start_simple.sh`：按 `Ctrl+C`
- 如果使用其他脚本：关闭对应的终端窗口，或运行：
  ```bash
  pkill -f realsense_camera_node
  pkill -f roscore
  ```
